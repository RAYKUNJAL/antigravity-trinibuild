## 🚨 Red Flags & Escalation

### Red Flag #1: CTR < 1% (Low Ad Performance)
```
Indicator: Fewer than 100 clicks from 10,000 impressions
Action:
1. Pause low-CTR ads immediately
2. Review ad copy: Is headline compelling?
3. Review creative: Does image/video grab attention?
4. Test new headline variation
5. Escalate: CAC will be > TT$50, unprofitable
Escalation to: Review hero messaging
```

### Red Flag #2: Signup Rate < 5% (Low Landing Page Performance)
```
Indicator: Only 5 signups from 100 clicks
Action:
1. Check if GA4 is firing correctly (debugging)
2. Check if button is visible above fold (mobile test)
3. Check scroll depth: Are users even scrolling?
4. Review hero CTA: Is it compelling?
5. A/B test: "Start Free" vs "Create Store"
Escalation to: Landing page redesign
```

### Red Flag #3: CAC > TT$50 (Unprofitable)
```
Indicator: Spending TT$500, only getting 10 signups
Action:
1. Check: Is this Week 1? (Allow higher CAC)
2. Pause all audiences except top 2 performers
3. Reduce budget to TT$500/week for testing only
4. Consolidate on 1 winning creative
5. Re-evaluate: Should we continue this channel?
Escalation to: Kill campaign or pause until optimization complete
```

### Red Flag #4: Store Completion < 10% (High Drop-off)
```
Indicator: 100 signups but only 10 stores created
Action:
1. Check if signup form is too long (> 5 fields?)
2. Check if category selection is blocking people
3. Check error messages in GA4 events
4. Simplify onboarding: Move non-critical fields to later
5. Send reminder email: "Complete your store setup"
Escalation to: Redesign signup flow
```
